AskForQuestion 工具参数设计

你希望工具的参数是什么结构？

A（推荐）

简洁参数结构

{
  "question": "请选择方案",
  "options": [
    {"id": "a", "label": "方案A", "desc": "..."},
    {"id": "b", "label": "方案B", "desc": "..."}
  ],
  "recommended": "a"
}
B

详细参数结构

{
  "title": "确认操作",
  "question": "请选择方案",
  "options": [
    {"id": "a", "label": "方案A", "desc": "...", "icon": "💡"},
    {"id": "b", "label": "方案B", "desc": "...", "icon": "🐧"}
  ],
  "recommended": "a",
  "allow_custom": true
}
