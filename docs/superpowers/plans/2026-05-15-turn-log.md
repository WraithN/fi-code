# Turn 级完整对话日志实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在每次 `run_one_turn` 结束后将完整回合上下文（LLM 输出、工具调用、参数、返回结果、Token 消耗、Turn 状态）以 JSON 形式持久化到 `~/.config/fi-code/logs/turns.jsonl`，并提供 `fi-code-cli logs` 命令以人类可读格式查看。

**Architecture:** 新增 `TurnLogger` 单例模块，通过 `tokio::sync::mpsc` 将 `TurnLogEntry` 异步发送给后台写入任务，避免阻塞 Agent 循环。CLI 命令独立读取 JSONL 文件并格式化输出。

**Tech Stack:** Rust, tokio, serde_json, clap, directories

```rust
// crates/core/src/agent/turn_logger.rs
use serde::Serialize;
use std::sync::OnceLock;
use tokio::sync::mpsc;

use crate::session::message::{Message, Part, TokenUsage};

#[derive(Debug, Clone, Serialize)]
pub struct ToolResultLog {
    pub tool_call_id: String,
    pub name: String,
    pub arguments: serde_json::Value,
    pub content: String,
    pub duration_ms: u64,
    pub is_error: bool,
}

#[derive(Debug, Clone, Serialize)]
pub struct TurnLogEntry {
    pub timestamp: String,
    pub session_id: String,
    pub turn_index: usize,
    pub finish_reason: Option<String>,
    pub token_usage: TokenUsage,
    pub content_blocks: Vec<Part>,
    pub tool_results: Vec<ToolResultLog>,
    pub messages_snapshot: Vec<Message>,
    pub wave_marker: Option<Part>,
    pub transition_reason: Option<String>,
    pub error: Option<String>,
}

/// 后台写入通道句柄
pub struct TurnLogger {
    tx: mpsc::Sender<TurnLogEntry>,
}

impl TurnLogger {
    pub fn global() -> &'static TurnLogger {
        static INSTANCE: OnceLock<TurnLogger> = OnceLock::new();
        INSTANCE.get_or_init(|| {
            let (tx, rx) = mpsc::channel::<TurnLogEntry>(1024);
            tokio::spawn(async move {
                Self::run_writer(rx).await;
            });
            TurnLogger { tx }
        })
    }

    pub fn log_turn(&self, entry: TurnLogEntry) {
        let _ = self.tx.try_send(entry);
    }

    async fn run_writer(mut rx: mpsc::Receiver<TurnLogEntry>) {
        use tokio::io::AsyncWriteExt;
        let Some(proj_dirs) = directories::ProjectDirs::from("", "", "fi-code") else {
            return;
        };
        let logs_dir = proj_dirs.config_dir().join("logs");
        let _ = tokio::fs::create_dir_all(&logs_dir).await;
        let path = logs_dir.join("turns.jsonl");
        let Ok(mut file) = tokio::fs::OpenOptions::new()
            .create(true)
            .append(true)
            .open(&path)
            .await
        else {
            return;
        };

        while let Some(entry) = rx.recv().await {
            let line = match serde_json::to_string(&entry) {
                Ok(s) => s,
                Err(_) => continue,
            };
            let _ = file.write_all(line.as_bytes()).await;
            let _ = file.write_all(b"\n").await;
        }
    }
}
```

具体位置（在已有代码的 `return` 或 `Ok(...)` 之前插入）：

1. **Line 350 附近** — `direct_output` 分支返回 `Ok(false)` 前
2. **Line 440 附近** — tool_use 但无 results 返回 `Ok(false)` 前
3. **Line 508 附近** — `tool_result` 分支返回 `Ok(true)` 前（在 `state.turn_count += 1` 之后）
4. **Line 574** — 函数末尾 `Ok(true)` 前

由于 `run_one_turn` 有多个提前返回点，最佳实践是：在函数末尾（`Ok(true)` 之前）统一记录，同时用 `std::mem::replace` 或函数提取避免在每个 return 前重复代码。

更简单的方式：在 `agent_loop` 中记录，因为 `agent_loop` 在每次 `run_one_turn` 返回后都能看到 `LoopState` 和 `TurnState`。但 `TurnState` 在 `run_one_turn` 内部，外部看不到。

所以还是在 `run_one_turn` 内部记录。由于 `run_one_turn` 内部 `turn` 变量存活到函数末尾，可以在函数最后、任何 return 之前统一记录。但当前代码有多个 `return Ok(false)` 提前退出点。

**方案**：在函数末尾统一记录，并修改提前返回点为 `result = Ok(false); break 'turn;` 风格，或用一个 `Drop` guard。

最简单的方案：在每个 `return` 前插入记录代码。由于只有 4 个返回点，重复是可以接受的。

插入的代码块（在每个 return 前）：

```rust
// 构造 TurnLogEntry
```

```rust
// crates/core/src/utils/turn_log_cli.rs
use std::path::PathBuf;
use anyhow::{Context, Result};
use serde_json::Value;

pub struct LogsOptions {
    pub limit: usize,
    pub follow: bool,
    pub session_filter: Option<String>,
    pub tool_filter: Option<String>,
    pub raw: bool,
}

pub fn default_log_path() -> PathBuf {
    directories::ProjectDirs::from("", "", "fi-code")
        .map(|d| d.config_dir().join("logs").join("turns.jsonl"))
        .unwrap_or_else(|| PathBuf::from(".config/fi-code/logs/turns.jsonl"))
}

pub fn run_logs_cli(options: LogsOptions) -> Result<()> {
    let path = default_log_path();
    if !path.exists() {
        anyhow::bail!("日志文件不存在: {}", path.display());
    }

    let content = std::fs::read_to_string(&path)
        .with_context(|| format!("读取日志文件失败: {}", path.display()))?;

    let lines: Vec<&str> = content.lines().collect();
    let filtered: Vec<Value> = lines
        .iter()
        .filter_map(|line| serde_json::from_str::<Value>(line).ok())
        .filter(|v| {
            if let Some(ref session) = options.session_filter {
                v.get("session_id")
                    .and_then(|s| s.as_str())
                    .map(|s| s.starts_with(session))
                    .unwrap_or(false)
            } else {
                true
            }
        })
        .filter(|v| {
            if let Some(ref tool) = options.tool_filter {
                v.get("tool_results")
                    .and_then(|arr| arr.as_array())
                    .map(|arr| arr.iter().any(|t| t.get("name").and_then(|n| n.as_str()) == Some(tool.as_str())))
                    .unwrap_or(false)
            } else {
                true
            }
        })
        .collect();

    let start = filtered.len().saturating_sub(options.limit);
    for entry in &filtered[start..] {
        if options.raw {
            println!("{}", serde_json::to_string_pretty(entry).unwrap());
        } else {
            print_formatted(entry);
        }
    }

    Ok(())
}

fn print_formatted(entry: &Value) {
    let turn_idx = entry.get("turn_index").and_then(|v| v.as_u64()).unwrap_or(0);
    let session_id = entry.get("session_id").and_then(|v| v.as_str()).unwrap_or("?");
    let ts = entry.get("timestamp").and_then(|v| v.as_str()).unwrap_or("?");
    let finish = entry.get("finish_reason").and_then(|v| v.as_str()).unwrap_or("?");
    let usage = entry.get("token_usage").cloned().unwrap_or_default();
    let prompt = usage.get("prompt_tokens").and_then(|v| v.as_u64()).unwrap_or(0);
    let completion = usage.get("completion_tokens").and_then(|v| v.as_u64()).unwrap_or(0);

    println!("═══════════════════════════════════════════════════");
    println!("Turn #{} | Session: {}... | {}", turn_idx, &session_id[..session_id.len().min(8)], ts);
    println!("Finish: {} | Tokens: {}↑ {}↓", finish, prompt, completion);
    println!("───────────────────────────────────────────────────");

    // LLM 输出
    if let Some(blocks) = entry.get("content_blocks").and_then(|v| v.as_array()) {
        for block in blocks {
            if let Some(text) = block.get("text").and_then(|v| v.as_str()) {
                if !text.trim().is_empty() {
                    println!("[LLM]\n{}", text);
                }
            }
        }
    }

    // 工具结果
    if let Some(results) = entry.get("tool_results").and_then(|v| v.as_array()) {
        for tr in results {
            let name = tr.get("name").and_then(|v| v.as_str()).unwrap_or("?");
            let duration = tr.get("duration_ms").and_then(|v| v.as_u64()).unwrap_or(0);
            let is_error = tr.get("is_error").and_then(|v| v.as_bool()).unwrap_or(false);
            let icon = if is_error { "❌" } else { "✅" };
            let args = tr.get("arguments").cloned().unwrap_or_default();
            let content = tr.get("content").and_then(|v| v.as_str()).unwrap_or("");

            println!("\n[Tool] {} | {} {}ms", name, icon, duration);
            println!("参数: {}", serde_json::to_string(&args).unwrap());
            println!("结果: {}", content.chars().take(500).collect::<String>());
        }
    }

    println!("═══════════════════════════════════════════════════\n");
}
```
